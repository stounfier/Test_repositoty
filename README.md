# Git & GitHub Basic Task

#Мета
Ознайомлення з базовими діями роботи з Git і GitHub: створення репозиторію, налаштування SSH-з’єднання, виконання комітів і публікація проєкту на GitHub.

#Зміст репозиторію
-  src - мейн проєкту
-  docs - документація проєкту
- `README.md` — опис проєкту  
- `test.txt` — тестовий файл для коміту 

#Інструкція з використання
1. Встанови Git на комп’ютер.  
2. Створи SSH-ключ і додай його до GitHub.  
3. Клонуй або створи репозиторій:
   bash
   git init
   git remote add origin git@github.com:stounfier/Test_repositoty.git 

#Функціонал
- Ініціалізація локального репозиторію (`git init`)
- Підключення до GitHub через SSH
- Додавання та відстеження файлів (`git add`, `git commit`)
- Надсилання змін у віддалений репозиторій (`git push`)
- Перегляд історії комітів (`git log`)
- Робота з гілками (`git branch`, `git merge`)
- Створення та оновлення README-файлу

#Виконано дії
1. Встановлено Git.  
2. Створено SSH-ключ і додано до GitHub.  
3. Створено локальний репозиторій і пов’язано з віддаленим через SSH.  
4. Виконано 2 коміти:  
   - `add test file`  
   - ` add main.py file`
   - ` update docs`
   - ` feature-update change`

#Використані інструменти
Робота виконувалася через Git Bash.
